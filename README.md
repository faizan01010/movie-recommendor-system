# 🎬 Movie Recommendation System

A Machine Learning based Movie Recommendation System built using **Streamlit** and **Cosine Similarity**.

This app recommends similar movies based on user-selected input and displays details using the **TMDB API**.

---

## 🔥 Features

- 🎯 Recommend similar movies
- ⭐ Shows movie rating
- 📅 Shows release year
- 🖼️ Movie posters from TMDB API
- 🎥 Trailer links
- ⚡ Clean Streamlit UI with hover effects

---

## 🧠 Tech Stack

- Python 🐍
- Streamlit 🎈
- Pandas 📊
- Scikit-learn 🤖
- Requests 🌐
- TMDB API 🎬
